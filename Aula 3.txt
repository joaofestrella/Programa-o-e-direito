a) Lei geral de proteção de dados é uma lei que surgiu com a finalidade de versar sobre dados, isto é caso haja o descumprimento desta lei, haverá punições. Essa lei tem haver com um ciclo de vida, ou seja, o dado deve ser excluído após seu uso.Principalmente quando se fala em dados pessoais, essa lei tem de atuar de maneira mais severa, devido a esse tipo de dado ser muito específico e revelar a intimidade da pessoa. É importante que se fiscalize se essa lei está sendo devidamente cumprida, pelo fato de se tratar da sensibilidade dos dados. Quem realiza essa fiscalização é o ministério público.Esse assunto envolve também as entidades de defesa do consumidor. 
Data center é um local físico que armazena máquinas de computadores, equipamentos de software e hardware. Ou seja, é um lugar em empresas que têm como função armazenar os equipamentos de informática. A função de um data center é levar os dispositivos para uma localização central e gerenciá-los. 
A relação do data centers com a lei geral de proteção de dados no Brasil é que a Lei geral de proteção de dados trata de impedir que haja violação aos dados e o datacenters armazena muitos dados nas máquinas de computadores e nos dispositivos de hardware e software e também não se sabe se os dados de uma empresa por exemplo são tratados no meio físico que são os data centers ou no meio digital. 
b) Formatos de arquivo aberto é uma técnica de informática que tem como função armazenar dados digitais, e pode ser usado sem restrições legais. Os objetivos dos formatos abertos são garantir o acesso a dados a longo prazo para minimizar as chances de insegurança jurídica ou restrições devido às especificações técnicas e também garantir a concorrência. 
Os formatos de arquivo privado são controlados por interesses privados, isto é, não são públicos. Nesse formato, caso o software pare de funcionar, os dados são perdidos devido a não ter outros sistemas para armazená-los. As especificações de domínio público se tornam ilegíveis pelo formato de arquivo proprietário.
b.1- Exemplo de formatos de arquivo aberto: ODT, ODS, ODP, CSV, SVG e PDF.
Exemplos de formatos de arquivo proprietário: DWG, SWF, DOC e PPT.
b.2-  Dados públicos são dados que se pode utilizar, compartilhar e acessar livremente. São considerados dados públicos os que são de interesse do coletivo e para serem acessados precisam de um sistema de privacidade. Os dados públicos podem ser inseridos em diferentes contextos. Esses contextos em que são inseridos são o contexto em que se tem informações e fatos públicos.
